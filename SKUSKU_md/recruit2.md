# 멋쟁이사자처럼 14기 리크루팅 페이지 (Recruit2) 기획안

이 문서는 10년차 PO 및 FE-Dev의 관점에서 기획된 `Recruit2` 페이지의 디자인, 애니메이션, 콘텐츠 명세서입니다. 전체 시스템은 반응형 웹으로 동작하며 최고 수준의 UI/UX 인터랙션을 제공합니다.

---

## 🎨 Global UI/UX & 디자인 가이드

**1. 기본 인터랙션 및 로딩**
- **Lenis Smooth Scroll (부드러운 스크롤):** 마우스 휠의 뚝뚝 끊기는 움직임을 관성(Inertia)이 적용된 부드러운 움직임으로 변환하여 모든 고급 인터랙션의 필수 기반을 다집니다.
- **Cinematic Preloader (시네마틱 프리로더):** 사이트 리소스가 로딩될 때까지 진행률을 보여주다가, 완료 시 검은 막(Curtain)이 걷히듯 메인 콘텐츠를 우아하게 등장시킵니다.

**2. 테마 전환 (Theme Transition / Background Color Swap)**
- **동작 방식:** 긴 스크롤 페이지의 분위기를 환기하기 위해 특정 섹션(Section 4: 모집 파트 이전)에 도달하면 body의 배경색과 글자색이 부드럽게 반전(Dark ↔ Light)됩니다.
- **디자인 원칙:** 검정(Dark)에서 흰색(Light) 테마로 전환되어 페이지 끝까지 Light 테마가 유지됩니다. 단, 눈의 피로를 막고 고급스러움을 유도하기 위해 **완전한 원색(#000000, #FFFFFF) 사용은 금지**합니다.

**3. 직군별 포인트 컬러 (Track Colors)**
명확한 시각적 구분을 위해 역할 배지나 프로필 아이콘 등에 아래 컬러를 공통 적용합니다.
- **DESIGN (기획/디자인):** `#A259FF`
- **FRONTEND (프론트엔드):** `#1ABCFE`
- **BACKEND (백엔드):** `#0ACF83`

## 4. 모집 파트 (Tracks)

**콘텐츠 및 메시지**
1. **공통 세션:** Git/GitHub, 서버 기본 형태, 바이브 코딩을 통한 빠른 아이디어 구현 (협업 기본기)
2. **기획/디자인:** 문제 정의, 사용자 흐름(UX), Figma 와이어프레임 & UI (1비율)
3. **프론트엔드:** HTML/CSS/Tailwind, JS 동적 요소, React 컴포넌트, TypeScript (2비율)
4. **백엔드:** Java & Spring Boot, CRUD API 설계, DB 관리, 클라이언트 통신 및 Security (2비율)

**✨ 애니메이션 (UI/UX)**
- **[Desktop] Accordion Gallery (아코디언 갤러리):** 가로 공간이 충분한 PC 환경에서는 파트별 카드(이미지)가 가로로 나열되며, 마우스 호버 시 해당 뷰가 부드럽게 가로로 확장되고 나머지 카드는 축소되는 인터랙션을 제공합니다.
- **[Mobile] Scroll Stacking Cards (스크롤 스태킹 카드) 추천도: ⭐⭐⭐⭐⭐:** 가로폭이 좁고 세로 스크롤 위주의 모바일 환경에서는 사용자가 스크롤을 내릴 때 각 파트 카드가 화면 상단에 차곡차곡 포개어 쌓이는(Sticky Stack) 애니메이션을 적용하여 100%의 가독성과 몰입감을 제공합니다.
